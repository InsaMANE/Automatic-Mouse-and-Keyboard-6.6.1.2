## 🎮 How to Use

1. **Record a Macro:** Click the "Record" button, perform your desired mouse and keyboard actions, then click "Stop."
2. **Edit the Macro:** Use the  editor to fine-tune timings, add loops, or insert conditions.

4. **Play the Macro:** Press the assigned hotkey to replay the recorded sequence.
